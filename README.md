# Description
Implémentation et comparaison d'algorithmes avec analyse de complexité temporelle.

## Fonctions Implémentées

1/.Lecture de fichier read_file() - Lit les valeurs entières d'un fichier  

2/.Comptage d'occurrences nombre_occurrences() - O(n²) - Méthode naïve  

nombre_occurrences_ameliore() - O(n) - Méthode optimisée  

3/.Algorithmes de tri selection_sort() - Tri par sélection - O(n²)  
merge_sort() - Tri par fusion - O(n log n)  

4/.Sauvegarde write_to_file() - Sauvegarde les données triées
